Добавь новый экран категорий продуктов который бы показывал список продуктов внутри каждой категории товара.

Переход на этот экран должен происходить из экрана списка продуктов. У каждой категории (справа) должна быть кнопка "Посмотреть все". 
По нажатию на эту категорию нужно запускать этот новый экран.

Сам экран из себя представляет вертикальный grid-список, строка каждого grid-списка содержит 2 элемента.
Каждый элемент это карточка продукта у которой есть: картинка, название, и цена

На экране есть кнопка Назад, которая должна вести пользователя на предыдущий экран со списком продуктов.

Данные продуктов пока сделай mock-данными.
