## Цель
Хочу сделать новую фичу.

## Описание
Добавь новый экран карточки продукта который открывается по нажатию на карточку продукта внутри экрана списка продуктов и категорий продукта.

Данный экрана должен содержать картинку товара, название товара, описание, характеристики.
При открытии товара пока грузяться данные, важно показывать пользователю Loader. В случае ошибки при получении данных,
нужно показывать состояние ошибки в виде отдельной View c описанием ошибки.

Экран должен содержать кнопку "Назад", которая бы возвращала его на основной экран со списком продуктов.

## Полученение данных продуктов

### Сетевой запрос к ktor
Для получения информации о продукте по id используй запрос GET http://10.0.2.2:8080/products/{id}
Где id (path, Long) — идентификатор товара

### Генерация моделей
Для генерации моделей запросов и ответов используй стандарт OpenApi JSON, он описан и доступен по
по ссылке http://localhost:8080/v3/api-docs
