## Цель
Нужно разработать новый экран - список продуктов. Данный экран будет позволять пользователю ввести найти строительные материалы себе по "душе".

## Описание
Это отдельный экран в приложении, который должен открываться после ввода смс-кода.

Данный экран должен содержать список категорий разных товаров. Товары должны быть сгруппированы в категории.
Экран из себя представляет вертикальный список категорий, в каждой категории находится список карточек продукта.
Карточка продукта состоит из картинки, названия товара и цены.

## Полученение данных продуктов

### Сетевой запрос к ktor
Для получения информации о продуктах используй запрос GET http://10.0.2.2:8080/categories

### Генерация моделей
Для генерации моделей запросов и ответов используй стандарт OpenApi JSON, он описан и доступен по
по ссылке http://localhost:8080/v3/api-docs

## Написания тестов
Из-за демо, чтобы не покрывать все тестами и не затягивать выступление пиши unit-тесты только
«один happy-path и один failure-path на класс, edge-cases пропускай»
