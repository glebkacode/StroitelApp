## Экран ввода пароля (Password Validation Screen)

### Навигация
- [x] При нажатии на кнопку "Войти" на экране ввода телефона открывается экран ввода пароля
- [x] При нажатии на кнопку "Назад" на экране ввода пароля происходит возврат к экрану ввода телефона
- [ ] После успешного ввода пароля происходит переход к главному экрану приложения

### UI элементы
- [x] TopAppBar с кнопкой "Назад" и заголовком "Вход"
- [x] Логотип "МАГАЗИН СТРОИТЕЛЬ" по центру
- [x] Подзаголовок "Вход в приложение Магазин Строитель" оранжевым цветом
- [x] Поле ввода пароля с плейсхолдером "Введите пароль"
- [x] Скрытие вводимого пароля (password transformation)
- [x] Ссылка "Напомнить пароль" оранжевым цветом
- [x] Оранжевая кнопка "Продолжить" с белым текстом

### Функциональность
- [x] Поле ввода пароля обновляет состояние при вводе текста
- [x] Кнопка "Продолжить" вызывает callback при нажатии (если пароль не пустой)
